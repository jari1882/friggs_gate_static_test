---
description: Stage, commit, and push changes
---

Stage all changes, create an appropriate commit message, and push to the current branch.

Use the following steps:
1. Run `git status` and `git diff` to see what changes have been made
2. Analyze the changes and create a descriptive commit message
3. Run `git add .` to stage all changes
4. Run `git commit -m "your generated message"` to commit with the generated message
5. Run `git push` to push to the remote repository
